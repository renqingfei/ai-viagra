# 丸子Ai 使用文档

> **版本:** 2.0.0  
> **作者:** 丸子团队  
> **更新日期:** 2026年1月

---

## 📖 简介

丸子Ai 是一款智能反馈工具插件，让 AI 在完成任务后等待您的反馈，帮助您掌控对话节奏。支持 Windsurf 账号管理、自动切换等高级功能。

---

## 🚀 快速开始

### 1. 安装插件

1. 下载 `AI-Echo.vsix` 文件
2. 打开 Windsurf/VSCode
3. 按 `Ctrl+Shift+P` 打开命令面板
4. 输入 `Install from VSIX`
5. 选择下载的 `.vsix` 文件
6. 重启编辑器

### 2. 激活插件

1. 点击左侧边栏的 **丸子Ai** 图标
2. 在「状态」页面输入激活码
3. 点击「🔑 激活」按钮
4. 激活成功后即可使用全部功能

## 📊 功能模块

### 状态页面

| 功能 | 说明 |
|------|------|
| **激活状态** | 显示当前激活码状态、剩余天数 |
| **公告** | 显示管理员发布的最新公告 |
| **服务状态** | 显示 MCP 服务运行状态 |
| **使用统计** | 对话次数、继续率等数据统计 |

### 账号管理

> ⚠️ **重要提示:** 使用账号管理功能前，请确保已开启代理/VPN

| 功能 | 说明 |
|------|------|
| **批量导入** | 支持批量导入 Windsurf 账号（格式：每行一个账号，邮箱+空格+密码） |
| **一键切换** | 无需重启即可切换 Windsurf 账号 |
| **积分显示** | 实时显示账号剩余积分和到期时间 |
| **自动切换** | 积分不足时自动切换到下一个有积分的账号 |
| **机器ID重置** | 重置 Windsurf 机器ID，绕过设备限制 |

#### 批量导入格式
```
user1@email.com password123
user2@email.com password456
user3@email.com password789
```

#### 自动切换设置
- **切换阈值:** 当积分低于此值时触发自动切换（默认 10）
- **检测间隔:** 每隔多少秒检测一次积分（默认 300 秒）
- **低积分提醒:** 当积分低于此值时弹出提醒（0 表示关闭）

### 历史记录

| 功能 | 说明 |
|------|------|
| **查看历史** | 查看所有对话记录 |
| **多选导出** | 点击选择多条记录，导出为 JSON 或 Markdown |
| **清空历史** | 一键清空所有历史记录 |

### 快捷短语

在反馈弹窗中快速插入常用短语，节省输入时间。

- 点击「➕ 添加短语」添加新短语
- 点击短语可在反馈弹窗中快速插入

### 模板

系统提示词模板，AI 会优先看到这些指令，用于引导 AI 行为。

- 点击「➕ 添加模板」添加新模板
- 勾选启用后，AI 将遵循模板中的指令

### 设置

| 设置项 | 说明 |
|--------|------|
| **主题** | 深色/浅色 |
| **弹窗位置** | AI聊天旁边（推荐）/ 代码区全屏 |
| **AI反馈高度** | 调整弹窗中 AI 摘要区域高度 |
| **用户反馈高度** | 调整弹窗中用户输入区域高度 |
| **字体大小** | 调整弹窗字体大小 |

## ❓ 常见问题

### Q: 为什么账号登录失败？
A: 请确保已开启代理/VPN，账号管理功能需要访问国外服务器。

### Q: 为什么自动切换不生效？
A: 请检查自动切换是否已启用，并确保切换阈值和检测间隔设置正确。

### Q: 为什么弹窗没有出现？
A: 
1. 确保使用的是思考模型（如 Claude 3.5）
2. 发送以下消息让 AI 调用工具：`请在每次回复结束时调用 infinite_dialog_feedback 工具`

### Q: 如何解绑设备？
A: 在「状态」页面点击「解绑」按钮。注意：解绑会扣除 3 天时间。

### Q: 如何重置机器ID？
A: 在「账号」页面底部点击「🔄 重置机器ID」按钮，重置后需要重启 Windsurf。
