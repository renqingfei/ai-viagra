# 丸子Ai - 智能反馈助手

让 AI 在完成任务后等待您的反馈，掌控对话节奏。

## ✨ 功能特点

- 🔄 **智能反馈** - AI 完成任务后自动弹窗等待您的指令
- 📋 **剪贴板图片** - 支持 Ctrl+V 粘贴截图
- 🖱️ **拖拽上传** - 拖拽文件到对话框
- 🎨 **深浅主题** - 支持深色/浅色主题切换
- 🔔 **提示音** - 6种系统提示音可选
- 📊 **数据统计** - 记录使用历史和统计
- 👤 **账号管理** - Windsurf账号一键切换
- 🆓 **开箱即用** - 无需复杂配置

## 🚀 支持的 IDE

- Windsurf

## 📦 安装方法

1. 下载 `AI-Echo.vsix` 文件
2. 打开 Windsurf，按 `Ctrl+Shift+P`
3. 输入 `Extensions: Install from VSIX`
4. 选择下载的 VSIX 文件
5. **重启 Windsurf**

## ⚙️ 配置项

在设置中搜索 `infiniteDialog`：

- `infiniteDialog.serverPort` - 服务端口（默认 3456）
- `infiniteDialog.autoConfigureRules` - 是否自动配置全局规则（默认 true）
